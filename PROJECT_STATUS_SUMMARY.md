# 📊 AttentionSync 项目状态总结

## 🎯 项目概述

**AttentionSync** 是一款开源的智能信息聚合平台，通过AI技术自动采集、转写、摘要和个性化推荐多源信息，帮助用户实现高效的信息消费。

**核心理念**: 让每个人用3分钟掌握一天的关键信息，永不错过重要机会。

- **缓存**: Redis
