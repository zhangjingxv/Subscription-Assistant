## 🚀 立即开始使用

### 快速启动 (3分钟内运行)

```bash
# 1. 运行启动脚本
./scripts/start.sh

# 2. 访问应用
# 前端界面: http://localhost:3000
# API文档: http://localhost:8000/docs

# 3. 登录系统
# 邮箱: admin@attentionsync.io
# 密码: admin123
```

### 配置AI功能
编辑 `.env` 文件，添加AI API密钥：
```env
ANTHROPIC_API_KEY=your_claude_api_key_here
# 或者
OPENAI_API_KEY=your_openai_api_key_here
```
